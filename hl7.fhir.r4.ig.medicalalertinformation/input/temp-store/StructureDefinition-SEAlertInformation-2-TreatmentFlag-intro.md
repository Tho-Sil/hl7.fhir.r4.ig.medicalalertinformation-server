# Introduktion

Denna profil används för att dokumentera och uppmärksamma pågående eller tidigare behandlingar som är av särskild betydelse för patientens vård och säkerhet. Det kan till exempel röra sig om läkemedelsbehandlingar, dialys, antikoagulantiabehandling eller annan behandling som kräver särskild uppmärksamhet vid vårdinsatser.

Syftet är att säkerställa att viktig information om patientens behandlingar är lättillgänglig och kan beaktas i alla vårdsituationer. Urvalet av behandlingar är baserat på Socialstyrelsens nationella specifikation för medicinsk uppmärksamhetsinformation.
